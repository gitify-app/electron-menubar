# example-menubar-hello-world

## Description

This is a simple Hello World menubar app that will just open a window that says "Hello World".

## Screenshot

![screenshot](./screenshot.png)

## Instructions

- Clone the repository.
- Run `pnpm install` from the root folder.
- Run `pnpm build` from the root folder.
- `cd` into this directory.
- Run `pnpm install` to install this example's dependencies.
- Run `pnpm start` from this directory to run app.
