# example-menubar-icon-animation

## Description

A menubar app that animates the tray icon by swapping images on an interval, useful for showing a "syncing" state while data is being fetched.

## Instructions

- Clone the repository.
- Run `pnpm install` from the root folder.
- Run `pnpm build` from the root folder.
- `cd` into this directory.
- Run `pnpm install` to install this example's dependencies.
- Run `pnpm start` from this directory to run app.
